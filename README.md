# HTML e CSS

Este repositório reúne exercícios e desafios feitos durante o curso de HTML5 e CSS3 do Curso em Vídeo, ministrado pelo professor Gustavo Guanabara.

## Por que esta pasta existe

A pasta `html-css` foi criada para documentar e organizar meu aprendizado em desenvolvimento web front-end. Ela serve como uma coleção de projetos práticos que mostram minha evolução em:

- Estruturação de páginas com HTML
- Estilização com CSS
- Organização de arquivos e pastas para estudos
- Exportação de trabalhos para uso futuro ou portfólio

## Como usar este repositório

1. Abra a pasta `html-css` no seu editor de código preferido.
2. Navegue pelas pastas `desafios` e `exercicios` para ver cada projeto.
3. Abra o arquivo `index.html` de cada exercício no navegador para visualizar o resultado.
4. Use os arquivos CSS relacionados para entender como cada layout e estilo foram construídos.

## Estrutura do repositório

- `desafios/` — projetos desenvolvidos como desafios práticos do curso.
- `exercicios/` — exercícios organizados por módulo e numerados conforme o conteúdo do curso.
- `README.md` — descrição geral do repositório e instruções de uso.

### Exemplo de organização

- `desafios/D001/index.html`
- `desafios/D009/pag-html.html`
- `exercicios/Módulo-01/EX001/index.html`
- `exercicios/Módulo-02/EX020/hover.html`

## O que você encontra aqui
